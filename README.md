# SQL Analysis: Revenue and Performance Report
<img width="633" height="355" alt="image" src="https://github.com/user-attachments/assets/915876de-cb58-4b03-9703-5ad76090a023" />

## Objective
Produce insights to support strategic growth and revenue improvements

## Tasks
- calculate revenue by treatment/doctor
- segment high-value patients
- simulate pricing changes- track changes over time
- flag and update anomalies

## Tools
- SQL (for querying, aggregating, and analyzing the dataset)
- BigQuery (for handling datasets and performing queries)
- ERD (for database schema)
- PowerPoint (for data presentation key insights for stakeholders)

## Deliver
- SQL scripts with comments
- revenue summary
- one-slide business insight pitch
